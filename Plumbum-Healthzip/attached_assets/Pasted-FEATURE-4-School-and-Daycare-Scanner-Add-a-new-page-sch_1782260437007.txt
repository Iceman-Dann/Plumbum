FEATURE 4 — School and Daycare Scanner
Add a new page /schools and a tab in the main navigation labeled "Schools." This lets parents check the lead risk of any school, daycare, or childcare facility near them — not just their home.
Build a GET /api/schools?lat=...&lng=... FastAPI endpoint that queries the Google Places API for schools and daycares within 1 mile of the given coordinates, returns their name, address, place type, and coordinates. For each result run the same risk scoring function used for home addresses and return the score alongside the place data. Store the Google Places API key in .env as GOOGLE_PLACES_API_KEY.
The /schools page layout: same nav as all pages. Hero section with headline "Is your child's school safe?" in 56px Playfair bold. Body text: "Lead exposure is especially dangerous for children. Check any school, daycare, or childcare facility in your area." Below it the same address search bar as the homepage, labeled "Enter your address or school name."
When searched, show a results list below — each result is a card with: institution name in 18px Playfair bold, address in 13px gray, institution type pill (SCHOOL / DAYCARE / CHILDCARE) in 11px all-caps, and the risk score displayed as a colored number — brick red for high, amber for moderate, green for low. A "View full report →" link on each card goes to /result?address=... with that institution's address.
Above the results list show a small Leaflet map with markers for each institution colored by risk score. Clicking a marker highlights the corresponding card below.
At the top of the results add a summary strip: "X of Y institutions near this address are HIGH RISK." If more than half are high risk make this strip brick red background white text. This is the line that gets screenshotted and shared.
Below the results add a section "What schools are required to do" — three paragraphs explaining the EPA's 3Ts program for schools, what parents can request from school administrators, and how to file a complaint if the school refuses to test.
